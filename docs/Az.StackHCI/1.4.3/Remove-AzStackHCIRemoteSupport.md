# Remove-AzStackHCIRemoteSupport

## SYNOPSIS
Removes Remote Support.

## SYNTAX

```
Remove-AzStackHCIRemoteSupport [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Remove-AzStackHCIRemoteSupport uninstalls Remote Support Deployment module.

## EXAMPLES

### EXAMPLE 1
```
Remove-AzStackHCIRemoteSupport
```
