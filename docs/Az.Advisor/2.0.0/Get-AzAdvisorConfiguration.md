# Get-AzAdvisorConfiguration

## SYNOPSIS
Retrieve Azure Advisor configurations and also retrieve configurations of contained resource groups.

## SYNTAX

### List (Default)
```
Get-AzAdvisorConfiguration [-SubscriptionId <String[]>] [-DefaultProfile <PSObject>] [-Break]
 [-HttpPipelineAppend <SendAsyncStep[]>] [-HttpPipelinePrepend <SendAsyncStep[]>] [-Proxy <Uri>]
 [-ProxyCredential <PSCredential>] [-ProxyUseDefaultCredentials] [<CommonParameters>]
```

### List1
```
Get-AzAdvisorConfiguration [-SubscriptionId <String[]>] -ResourceGroupName <String>
 [-DefaultProfile <PSObject>] [-Break] [-HttpPipelineAppend <SendAsyncStep[]>]
 [-HttpPipelinePrepend <SendAsyncStep[]>] [-Proxy <Uri>] [-ProxyCredential <PSCredential>]
 [-ProxyUseDefaultCredentials] [<CommonParameters>]
```

## DESCRIPTION
Retrieve Azure Advisor configurations and also retrieve configurations of contained resource groups.

## EXAMPLES

### EXAMPLE 1
```
Get-AzAdvisorConfiguration
```

### EXAMPLE 2
```
Get-AzAdvisorConfiguration -ResourceGroupName lnxtest
```
